# Templates

- `project-hub.md` — copy into a project folder as `project.md`
- `weekly-checklist-template.md` — copy and rename to `YYYYMMDD-weekly-review.md`
- `todo-format.md` — quick reference for tags/format
- `daily-checkin-template.md` — lightweight daily capture + Top 3
- `waitingfor-report-template.md` — periodic report format for open #waitingfor items
